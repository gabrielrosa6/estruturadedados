Lista list
ordenada · mutável · permite duplicatas · acesso por índice

A lista é a estrutura de dados mais versátil do Python. Ela armazena uma sequência ordenada de elementos de qualquer tipo — inteiros, strings, outras listas, objetos. Por ser mutável, você pode adicionar, remover e modificar elementos a qualquer momento. É equivalente a um array dinâmico em outras linguagens, sendo a escolha padrão quando você precisa de uma coleção que cresce e muda ao longo do tempo.

Ordenada Mutável Duplicatas Indexável Hashable
Operações
list.append(x)

Adiciona o elemento x ao final da lista. É a forma mais comum de inserção — rápida e direta, sem necessidade de especificar posição.

list.insert(i, x)

Insere o elemento x na posição i, deslocando os elementos seguintes para a direita. Útil quando a posição importa.

list.remove(x)

Remove a primeira ocorrência do valor x encontrado na lista. Lança ValueError se o valor não existir.

list.pop()

Remove e retorna o último elemento da lista. Pode receber um índice opcional: pop(i) remove o elemento na posição i.

list.sort() & list.reverse()

sort() ordena a lista in-place (modifica a própria lista, sem criar uma cópia). reverse() inverte a ordem dos elementos.

Saída
Lista vazia. Adicione elementos acima.
Estado atual da lista
lista vazia
Exemplo de uso
# Criar e manipular uma lista frutas = ["maçã", "banana", "laranja"] frutas.append("uva") # adiciona ao final frutas.insert(1, "pera") # insere no índice 1 frutas.remove("banana") # remove por valor ultima = frutas.pop() # remove e retorna o último frutas.sort() # ordena alfabeticamente # Acesso por índice primeira = frutas[0] ultima = frutas[-1]
Pilha stack (list)
LIFO — Last In, First Out · o último a entrar é o primeiro a sair

A pilha é uma estrutura de dados com acesso restrito: você só pode inserir ou remover pelo topo. O Python não tem um tipo nativo dedicado para pilhas — a list comum cumpre esse papel com eficiência. O princípio LIFO (Last In, First Out) aparece naturalmente em editores de texto (desfazer ações), na pilha de chamadas de funções, e em algoritmos de busca em profundidade.

LIFO Acesso O(1) no topo Acesso por índice
Operações
push — list.append(x)

Empilha um novo elemento no topo da pilha. Corresponde ao append() da lista. Operação O(1) — tempo constante.

pop — list.pop()

Remove e retorna o elemento do topo. O elemento que entrou por último é o primeiro a ser removido. Lança IndexError se a pilha estiver vazia.

peek — list[-1]

Consulta o elemento do topo sem removê-lo. Em Python, usa-se o índice -1 para acessar o último elemento.

Saída
Pilha vazia.
Pilha — topo ao centro
pilha vazia
Exemplo de uso
# Pilha com list — LIFO pilha = [] # Push — empilha no topo pilha.append("primeiro") pilha.append("segundo") pilha.append("terceiro") # Peek — vê o topo sem remover topo = pilha[-1] # "terceiro" # Pop — desempilha do topo pilha.pop() # retorna "terceiro" pilha.pop() # retorna "segundo"
Fila queue (deque)
FIFO — First In, First Out · o primeiro a entrar é o primeiro a sair

A fila segue o princípio FIFO (First In, First Out) — como uma fila de banco. Elementos entram pelo final e saem pela frente. O Python oferece collections.deque como implementação eficiente: inserção e remoção em ambas as extremidades são O(1). Usar uma list simples como fila é ineficiente, pois pop(0) tem custo O(n).

FIFO O(1) nas pontas Acesso aleatório eficiente
Operações
enqueue — deque.append(x)

Insere um elemento no final da fila. Quem entra por último terá que esperar mais para ser atendido.

dequeue — deque.popleft()

Remove e retorna o elemento da frente da fila — aquele que está esperando há mais tempo. O(1) com deque, O(n) com list.

Saída
Fila vazia.
Fila — esquerda sai, direita entra
fila vazia
Exemplo de uso
from collections import deque fila = deque() # Enqueue — entra pelo final fila.append("Alice") fila.append("Bruno") fila.append("Carla") # Ver quem está na frente frente = fila[0] # "Alice" # Dequeue — sai pela frente fila.popleft() # retorna "Alice"
Tupla tuple
ordenada · imutável · permite duplicatas · hashable

A tupla é como uma lista que não pode ser modificada após sua criação. Sua imutabilidade traz vantagens importantes: ocupa menos memória, é mais rápida, e por ser hashable pode ser usada como chave de dicionário. Use tuplas para representar dados que não devem mudar — coordenadas, configurações, registros de banco de dados, retornos com múltiplos valores.

Ordenada Imutável Duplicatas Hashable Indexável
Operações
Criar tupla

Tuplas são criadas com parênteses ou apenas com vírgulas. Uma vez criada, não pode ser alterada — não há append, remove ou sort.

tuple.count(x)

Conta quantas vezes o valor x aparece na tupla. Uma das poucas operações de leitura disponíveis.

Acesso por índice

tupla[0] acessa o primeiro elemento, tupla[-1] o último. O acesso por índice é permitido — só a modificação não é.

Saída
Tupla vazia. Adicione elementos acima.
Estado atual da tupla
tupla vazia
Exemplo de uso
# Tuplas representam dados fixos ponto = (10.5, -3.2) rgb_azul = (0, 120, 255) usuario = ("Ana", 28, "SP") # Acesso por índice (leitura) x = ponto[0] # 10.5 y = ponto[1] # -3.2 # Unpacking nome, idade, cidade = usuario # Usar como chave de dicionário mapa = {ponto: "origem"}
Dicionário dict
mapeamento chave → valor · chaves únicas · Python 3.7+ preserva inserção

O dicionário é a estrutura de mapeamento central do Python. Ele associa chaves únicas a valores — como um índice de livro. A busca por chave é extremamente rápida (O(1) em média) graças à implementação com tabela hash. É usado para representar objetos, configurações, contagens, agrupamentos e qualquer situação onde você precisa acessar dados por nome em vez de posição.

Mapeamento Mutável Chaves duplicadas Busca O(1) Ordenado por inserção
Operações
dict[chave] = valor

Insere um novo par ou atualiza o valor de uma chave existente. Se a chave já existe, o valor é substituído silenciosamente.

del dict[chave]

Remove o par chave-valor correspondente. Lança KeyError se a chave não existir.

dict.get(chave)

Retorna o valor associado à chave. Ao contrário de dict[k], não lança erro se a chave não existir — retorna None ou um padrão definido.

Visões do dicionário

keys() retorna todas as chaves, values() todos os valores, items() os pares (chave, valor) — todos como visões iteráveis do dicionário.

Saída
Dicionário vazio. Insira uma chave e valor acima.
Estado atual do dicionário
dicionário vazio
Exemplo de uso
# Representar um objeto com dicionário aluno = { "nome": "Maria", "nota": 9.5, "turma": "B" } # Acesso seguro com get() nota = aluno.get("nota", 0) # Atualizar e remover aluno["nota"] = 10 del aluno["turma"] # Iterar sobre os pares for chave, valor in aluno.items(): print(f"{chave}: {valor}")
Conjunto set
sem duplicatas · não ordenado · operações matemáticas de conjuntos

O conjunto armazena elementos únicos e não ordenados. Ele implementa o conceito matemático de conjunto e suporta operações como união, interseção e diferença de forma natural e eficiente. É a escolha ideal para eliminar duplicatas de uma coleção, verificar pertencimento rapidamente (O(1)), e realizar cruzamentos entre grupos de dados.

Operações Matemáticas
A | B — União

Retorna todos os elementos que estão em A, em B, ou em ambos. Nenhum elemento é repetido.

A & B — Interseção

Retorna somente os elementos que existem em ambos os conjuntos simultaneamente.

A - B — Diferença

Retorna os elementos que estão em A mas não estão em B. Ordem importa: A - B ≠ B - A.

A ^ B — Diferença Simétrica

Retorna elementos que estão em A ou em B, mas não em ambos — o inverso da interseção.
